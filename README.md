# Project Description

FitLife is a fitness club web application that allows users to:

register and log in,

book fitness and wellness services,

manage their reservations,

purchase membership plans,

calculate daily calories and macronutrients (BMR & BZHU).

The project is built as a full-featured website with backend logic, database
integration, and an interactive user interface.
